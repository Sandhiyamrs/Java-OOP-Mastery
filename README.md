# Java-OOP-Mastery

A compact repository showing Object-Oriented Programming concepts in Java:
- Classes & objects
- Inheritance & polymorphism
- Abstract classes & interfaces
- Encapsulation
- Composition & simple design patterns

## How to run

javac -d out src/main/java/com/example/oop/*.java src/main/java/com/example/oop/model/*.java src/main/java/com/example/oop/service/*.java src/main/java/com/example/oop/util/*.java
java -cp out com.example.oop.App
